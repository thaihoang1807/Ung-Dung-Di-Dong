

## Vai trò
- **Thành viên phụ trách giao diện chính của hệ thống (Frontend UI).**
- Chịu trách nhiệm thiết kế, lập trình và kết nối các chức năng hiển thị, tương tác người dùng trong ứng dụng Flutter.

## Nhiệm vụ chi tiết

| STT | Hạng mục | Mô tả công việc |
|-----|-----------|----------------|
| 1.1 | **Đăng ký, Đăng nhập** | Xây dựng giao diện người dùng cho quá trình đăng ký và đăng nhập tài khoản, kết nối với Firebase Authentication để xác thực người dùng. |
| 1.2 | **Trang chủ (Dashboard & Danh sách cây)** | Thiết kế trang chính của ứng dụng, hiển thị danh sách cây cảnh, thông tin tổng quan và các tính năng nhanh. |
| 1.3 | **Trang Thêm / Xoá / Sửa thông tin cây** | Tạo giao diện cho việc thêm cây mới, chỉnh sửa hoặc xóa thông tin cây có sẵn, liên kết dữ liệu với Firestore Database. |
| 1.4 | **Trang Ghi nhật ký chăm sóc** | Thiết kế giao diện cho người dùng ghi chép quá trình chăm sóc cây (tưới nước, bón phân, cắt tỉa…). |
| 1.5 | **Trang Thư viện ảnh của cây** | Xây dựng giao diện hiển thị ảnh chụp cây cảnh theo từng mục, kết nối với Firebase Storage để tải ảnh. |
| 1.6 | **Trang Cài đặt** | Cung cấp tùy chọn cấu hình tài khoản, thông báo, giao diện ứng dụng và đồng bộ dữ liệu. |

## Công nghệ sử dụng
- **Ngôn ngữ:** Dart  
- **Framework:** Flutter  
- **IDE:** Visual Studio Code / Android Studio  
- **Backend liên kết:** Firebase SDK (Authentication, Firestore, Storage, FCM)  
- **Công cụ quản lý mã nguồn:** Git & GitHub  

## Kết quả mong đợi
- Giao diện thân thiện, trực quan, dễ thao tác.
- Kết nối ổn định với Firebase để đảm bảo dữ liệu thời gian thực.
- Hỗ trợ tốt cho các tính năng IoT hiển thị dữ liệu cảm biến và điều khiển từ xa.
